## 模型类序列化器

在serializers.py中定义一个模型类序列化器

```python
class ProjectModelSerializer(serializers.ModelSerializer):

    class Meta:
        #1.指定参考哪一个模型类来创建
        model = Projects
        #2.指定为模型类的哪些字段来生成序列化器
        fields = '__all__'
```

### 指定全部字段校验

在终端中使用ipython实例化，发现会自动生成序列化器
